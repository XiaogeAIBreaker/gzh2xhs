# 小红书爆款封面设计分析

基于 `/imgs` 目录下8张小红书热门封面的分析结果，按照实际封面类型建立A-H型分类体系。

## 封面类型分类体系 (A-H型)

### A型 - 攻略指南型 (image8.png 新疆旅游攻略)
**适用内容**: 旅游攻略、地点推荐、路线规划
**核心特征**:
- 左侧真实照片 + 右侧文字信息条布局
- 地理位置标识📍作为视觉锚点
- 详细信息: 时间、价格、介绍文字
- 配色: 温暖橙黄色调 - 对应 `TEMPLATE_COLORS.A` {bg: "#FAFAFA", text: "#1A1A1A", accent: "#FF6B35"}
- 字体层次: 地点名28px、详情16px

### B型 - 情绪共鸣型 (image7.png 考研失败 & image5.png 累成狗)
**适用内容**: 个人经历、情感话题、励志内容
**核心特征**:
- 超大黑体字占画面60-70%
- 关键情绪词用彩色椭圆高亮
- 配色: 对应 `TEMPLATE_COLORS.B` {bg: "#FFFFFF", text: "#1A1A1A", accent: "#4ECDC4"}
- 装饰性几何线条和符号
- 字体: 主标题120px粗体、副标题无或很小

### C型 - 真人代言型 (image6.png 新房软装)
**适用内容**: 专业知识分享、个人IP展示、产品推荐
**核心特征**:
- 右侧真人照片占50%画面建立信任
- 左侧专业文字信息 + 彩色特色标签
- 实际使用场景背景
- 配色: 粉蓝撞色 (#FF69B4, #4169E1)
- 突出个人专业性和可信度

### D型 - 表情包趣味型 (image5.png 橘猫)
**适用内容**: 轻松娱乐、日常吐槽、幽默内容
**核心特征**:
- 可爱动物/表情包元素居中
- 蓝色装饰边框 (#4ECDC4)
- 大字标题 + 趣味视觉元素结合
- 轻松幽默的整体氛围
- 背景: 浅灰纹理 (#F8F9FA)

### E型 - 知识图解型 (image4.png 教师资格证)
**适用内容**: 考试指南、教育内容、专业知识
**核心特征**:
- 蓝紫渐变背景 (#6C7CE7 → #A8E6CF)
- 规整的信息层级结构
- 分点列举的详细内容
- 图标装饰和卡片式信息块
- 专业教育风格，权威感强

### F型 - 省钱攻略型 (image3.png 省钱攻略)
**适用内容**: 生活技巧、实用攻略、省钱方法
**核心特征**:
- 卡通插画人物元素
- 温暖米黄色背景 (#FFF8DC)
- 分条目的实用技巧列表
- 圆角卡片式信息块
- 生活化、亲民的视觉风格

### G型 - 笔记手账型 (image2.png 小红书海报)
**适用内容**: 学习笔记、知识总结、经验分享
**核心特征**:
- 笔记本纸张质感背景
- 手写风格装饰元素
- 黄色荧光笔高亮效果 (#FFEB3B)
- 红色圆形标识重点 (#F44336)
- 学习感和亲切感并重

### H型 - 动态创意型 (image1.png 小红书爆款)
**适用内容**: 创意设计、视觉冲击、新颖概念
**核心特征**:
- 撕纸效果和动态视觉元素
- 大喇叭等创意图形装饰
- 红黑强对比配色 (#FF4444, #1A1A1A)
- 波普艺术风格
- 极强的视觉冲击力

## 内容→类型匹配规则

### 判定优先级 (从高到低)
1. **旅游地点关键词** → A型攻略指南
2. **情感困扰/个人经历** → B型情绪共鸣
3. **专业技能/产品推荐** → C型真人代言
4. **日常吐槽/轻松话题** → D型表情包趣味
5. **考试教育/知识干货** → E型知识图解
6. **生活技巧/省钱方法** → F型省钱攻略
7. **学习笔记/经验总结** → G型笔记手账
8. **创意设计/视觉概念** → H型动态创意

### 关键词映射表
- **A型**: 旅游、攻略、路线、景点、住宿、交通
- **B型**: 失败、焦虑、困难、逆袭、情绪、感受
- **C型**: 装修、科普、推荐、专业、技巧、产品
- **D型**: 日常、搞笑、吐槽、轻松、有趣、表情
- **E型**: 考试、证书、教育、指南、知识、学习
- **F型**: 省钱、优惠、攻略、方法、技巧、生活
- **G型**: 总结、笔记、经验、分享、整理、复盘
- **H型**: 创意、设计、视觉、新颖、冲击、艺术

## 技术实现标准

### 画布规格（与代码同步）
- **标准尺寸**: 1080×1440 (3:4比例) - 定义于 `APP_CONSTANTS.CARD_SIZE`
- **字体系统**: 配置于 `APP_CONSTANTS.EMOJI_FONTS` 数组
  - 支持emoji的字体栈: "Apple Color Emoji", "Segoe UI Emoji", "Noto Color Emoji"
  - 中文字体: "PingFang SC", "Microsoft YaHei", "Arial"
- **色彩空间**: sRGB，确保对比度≥WCAG AA标准

### 字体层次标准
- **A型**: 地点名36px、时间24px、介绍18px
- **B型**: 主标题120-150px、无副标题
- **C型**: 标题48px、标签28px、介绍20px
- **D型**: 标题80px、装饰元素适中
- **E型**: 标题60px、小标题32px、正文24px
- **F型**: 标题54px、技巧条目20px
- **G型**: 标题72px、正文28px、高亮文字
- **H型**: 标题100px、强视觉冲击

### 布局密度控制（配置化标准）
- **信息密度**: 文字20-35%，留白和装饰65-80%
- **覆盖度标准**: `PROMPT_CONSTANTS.LAYOUT.CONTENT_COVERAGE_MIN`-`CONTENT_COVERAGE_MAX` (55-75%)
- **字符限制**: 每行≤`PROMPT_CONSTANTS.LAYOUT.MAX_CHARS_PER_LINE` (14个汉字)
- **视觉层次**: 最多3层，避免信息过载
- **焦点管理**: 单一视觉重心，Z字形阅读路径
