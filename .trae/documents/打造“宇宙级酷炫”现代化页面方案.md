## 页面目标

- 打造拥有粒子/渐变流动背景、精致微交互、滚动视差、顺滑页面过渡、玻璃拟态/霓虹风的现代页面
- 保证主流浏览器兼容、响应式适配、强烈但协调的视觉冲击力

## 现有基础

- 动效与风格：已使用 `framer-motion` 与 Tailwind，并有玻璃拟态/霓虹样式（如 `glass-card`、`shadow-neon`）
    - 参考：`src/app/page.tsx:13-35` 路由进入动效、`src/app/globals.css:82-91` 玻璃拟态与霓虹阴影
- 背景：已有宇宙背景（星空/极光/网格/噪声）
    - 参考：`src/components/CosmicBackground.tsx:1-9`，`src/app/layout.tsx:21-27`

## 页面结构

- Hero：霓虹标题、动态背景、3D 轻微漂浮、CTA 按钮
- Features：卡片网格，悬停倾斜与高光，微交互细节
- Showcase（Parallax）：多层分区元素随滚动产生视差与分层动画
- Tech/Design：颜色层次展示、图标按钮风格演示（科技感）
- CTA：强光晕聚焦、顺滑滚动至顶部/路由跳转过渡

## 关键效果实现

### 动态背景（粒子/渐变二选一或可切换）

- 默认使用现有极光/星空/网格组合，强化层次与运动（CSS + Tailwind 动画）
- 可选新增粒子背景：轻依赖方案优先（Canvas 简易粒子）或库方案（tsParticles），提供开关与 `prefers-reduced-motion` 降级

### 精致微交互

- 悬停倾斜：根据鼠标位置对卡片进行 `transform` 倾斜与高光（使用 `.tilt-3d` 与 `framer-motion`）
- 按钮高光/涟漪：复用 `btn-glow`，增强按压涟漪与磁吸效果（按钮靠近鼠标微移）
- 细节动画：图标轻微漂浮（`animation: float`）、元素入场弹性

### 滚动视差

- 使用 `framer-motion` 的 `useScroll/useTransform` 在各分层上实现差异移动与淡入
- 将背景层、前景层与装饰层分别设定不同滚动速率（保持 GPU 友好，仅用 `transform`）

### 页面过渡动画

- 在 App Router 中增加客户端布局包装，使用 `AnimatePresence` 对路由切换淡入淡出/位移
- Key 使用 `usePathname` 保证每次路由切换动画一致顺滑

### 玻璃拟态/霓虹风格

- 统一使用现有 `glass-card`、`shadow-neon`、`glow`，新增 `NeonHeading` 与 `FrostedPanel` 组件以复用
- 光晕与高光方向与鼠标位置联动，提高质感

### 响应式与色彩层次

- 栅格：`grid` 在 `sm/md/lg/xl` 断点递进，Hero 与 Showcase 在移动端简化动画幅度
- 字体：`clamp()` 实现流式排版；色彩按 `neon.blue/pink` 主次层级分配，对比度达标

### 图标与按钮科技感

- 继续使用 `lucide-react`，图标带小型内发光与漂浮；按钮具磁吸与涟漪，CTA 强光晕

## 兼容与性能

- 兼容：Chrome/Edge/Firefox/Safari，`backdrop-filter` 不支持时自动回退为半透明背景
- 动画降级：尊重 `prefers-reduced-motion`，粒子密度与振幅降级；滚动视差在低性能设备自动关闭
- 性能：只用 `transform/opacity`，`will-change` 仅在必要元素上；懒加载与 `IntersectionObserver` 控制入场；避免大面积重阴影

## 验收标准

- 动态背景：极光/星空/网格流动，或粒子背景可开关
- 微交互：卡片悬停倾斜与高光、按钮涟漪与磁吸
- 视差：滚动时各层元素有差异位移与淡入
- 过渡：路由跳转具顺滑淡入/位移动效
- 风格：全局玻璃拟态与霓虹光效统一、色彩层次鲜明
- 兼容：四大主流浏览器正常显示；不支持特性自动回退
- 响应式：手机/平板/桌面布局与动效幅度合理
- 性能：仅 `transform/opacity` 动画，页面交互顺畅，尊重无动画偏好
